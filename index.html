Understanding Color Roles

Monochromatic palettes use variations of a single hue, creating a harmonious scale from lightest to darkest. The roles are:

  • • Lightest — Background colors, page base
  • • Light — Card surfaces, elevated elements
  • • Base — Your primary brand color, tested for accessibility
  • • Dark — Text on light backgrounds
  • • Darkest — Headings, high emphasis text

Other schemes (Analogous, Complementary, etc.) use different hues to create contrast and visual interest. The role labels help you understand how to use each color in your design system.
